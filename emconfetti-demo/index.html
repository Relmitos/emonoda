    emconfetti-demo

    Описание¶

    emconfetti-demo - команда для тестирования оповещений об обновлениях торрентов, настроенных для emupdate в конфигурационной секции ёconfetti. Запущенная без опций, она отсылает демо-сообщения с помощью всех доступных плагинов.

    Опции¶

    Info

    Кроме собственных опций, указанных ниже, команда поддерживает еще и общий стандартный набор (например, --help и --config). За подробностями обратитесь на страницу config.

    • -y, --only-confetti <plugin1 plugin2 ...>

      • Отсылать только оповещения для указанного подмножества плагинов от всех настроенных в секции confetti.

    • -x, --exclude-confetti <plugin1 plugin2 ...>

      • Включить все плагины из confetti, кроме перечисленных в этой опции.
